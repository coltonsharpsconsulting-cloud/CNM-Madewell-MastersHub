This suite contains four independent, aligned assessments for my three-week cells unit — a diagnostic to surface prior knowledge, a formative check to drive in-the-moment intervention, a summative assessment of the unit objectives, and an authentic performance task. Each lives on its own page (linked below) and is built around the same essential idea — structure determines function — with differentiation tied to the learners in my SLO 1 classroom snapshot.

Privacy note: Any student references (e.g., the focus student “Mateo”) are de-identified composites from my SLO 1 profile, used to show differentiation. No real student is identifiable.
